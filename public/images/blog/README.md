# Blog images

Post images live here (jpg or webp).

Rules:
- Keep every image under 200kb. Compress before adding (squoosh.app works well).
- Use descriptive kebab-case names: `jackson-food-trucks.jpg`, not `IMG_4032.jpg`.
- Hero images look best at 1280x720 or wider 16:9.
- Reference them in post frontmatter as `/images/blog/<filename>`.
